HamroBank is a minimal banking system integrated with a powerful cheque signature forgery detection using Siamese Neural Network.

## ⚙️ Tech Stack

**Frontend**

- NextJS
- TypeScript
- TailwindCSS

**Backend**

- Python
- Django + SQLite
- JSONWebTokens

**Machine Learning**

- Tensorflow
- MNOP :) (Matplotlib, Numpy, OpenCV, Pandas)
- Jupyter Notebooks
